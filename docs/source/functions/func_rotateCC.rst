rotateCC
========

.. doxygenfunction:: rotateCC
   :project: FEHRobot

Examples
-------------------

.. code-block:: c++
   :linenos:

   rotateCC(40.0, 75.0);

Rotates the robot 75 degrees counter-clockwise with motors at 40% power.
